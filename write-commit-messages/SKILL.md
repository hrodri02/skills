## Rules
- Subject line: ≤ 72 characters. Count before finalizing — if over, shorten the scope
  name, pick a shorter verb, or drop lower-priority words.
- Use conventional commits format: `type(scope): imperative subject`
- Write as many commits as the diff warrants; don't combine unrelated changes.
